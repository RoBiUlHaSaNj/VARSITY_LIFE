Final Keyword in Java

The final keyword in Java indicates that something cannot be changed. It can be used with variables, methods, and classes:

  • Final Variables: Once initialized, their values cannot be changed.
  • Final Methods: Cannot be overridden by subclasses.
  • Final Classes: Cannot be inherited by other classes.
final int MAX_VALUE = 100;
MAX_VALUE = 200; // Compilation error

Inheritance in Java - extends Keyword

The extends keyword creates a subclass that inherits fields and methods from a superclass:

// Superclass
class Animal {
    public void eat() {
        System.out.println("This animal eats food.");
    }
}

// Subclass
class Dog extends Animal {
    public void bark() {
        System.out.println("The dog barks.");
    }
}

Static Keyword in Java

The static keyword allows certain members to belong to the class rather than any object instance. Key uses include:

  • Static Variables: Shared across all instances.
  • Static Methods: Can be called without creating an instance.
  • Static Blocks: Initialize static variables.
  • Static Nested Classes: Can access static members of the outer class.
public class University {
    static String universityName = "Global University";
    static int totalStudents = 0;
    
    static class UniversityInfo {
        void display() {
            System.out.println("University: " + universityName);
        }
    }
}
