SQL Concepts

1. Data Definition Language (DDL)

Used to define and modify database structure.

CommandSyntax Example
CREATE DATABASECREATE DATABASE database_name;
DROP DATABASEDROP DATABASE database_name;
CREATE TABLECREATE TABLE table_name (column1 datatype, column2 datatype, ...);
ALTER TABLEALTER TABLE table_name ADD column_name datatype;
ALTER TABLE table_name DROP COLUMN column_name;
ALTER TABLE table_name MODIFY column_name datatype;
DROP TABLEDROP TABLE table_name;
TRUNCATE TABLETRUNCATE TABLE table_name;

2. Data Manipulation Language (DML)

Used to manage data within tables.

CommandSyntax Example
INSERTINSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);
SELECTSELECT column1, column2 FROM table_name;
SELECT * FROM table_name;
UPDATEUPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;
DELETEDELETE FROM table_name WHERE condition;

3. Data Query Language (DQL)

Used to query data.

CommandSyntax Example
SELECTSELECT column1, column2 FROM table_name WHERE condition;
WHERE ClauseSELECT * FROM table_name WHERE column_name = value;
ORDER BYSELECT * FROM table_name ORDER BY column_name ASC/DESC;
GROUP BYSELECT column_name, COUNT(*) FROM table_name GROUP BY column_name;
HAVINGSELECT column_name FROM table_name GROUP BY column_name HAVING condition;

4. Data Control Language (DCL)

Used to control access to data.

CommandSyntax Example
GRANTGRANT privilege ON table_name TO user;
REVOKEREVOKE privilege ON table_name FROM user;

5. Transaction Control Language (TCL)

Used to manage transactions in a database.

CommandSyntax Example
COMMITCOMMIT;
ROLLBACKROLLBACK;
SAVEPOINTSAVEPOINT savepoint_name;
SET TRANSACTIONSET TRANSACTION READ WRITE;

6. Constraints

Used to specify rules for data in a table.

ConstraintSyntax Example
NOT NULLCREATE TABLE table_name (column_name datatype NOT NULL);
UNIQUECREATE TABLE table_name (column_name datatype UNIQUE);
PRIMARY KEYCREATE TABLE table_name (column_name datatype PRIMARY KEY);
FOREIGN KEYCREATE TABLE table_name (column_name datatype, FOREIGN KEY (column_name) REFERENCES another_table(column_name));
CHECKCREATE TABLE table_name (column_name datatype CHECK (condition));
DEFAULTCREATE TABLE table_name (column_name datatype DEFAULT value);

7. Joins

Used to retrieve data from multiple tables.

TypeSyntax Example
INNER JOINSELECT columns FROM table1 INNER JOIN table2 ON table1.col = table2.col;
LEFT JOINSELECT columns FROM table1 LEFT JOIN table2 ON table1.col = table2.col;
RIGHT JOINSELECT columns FROM table1 RIGHT JOIN table2 ON table1.col = table2.col;
FULL JOINSELECT columns FROM table1 FULL JOIN table2 ON table1.col = table2.col;

8. Aggregate Functions

Used to perform calculations on data.

FunctionSyntax Example
COUNTSELECT COUNT(column_name) FROM table_name;
SUMSELECT SUM(column_name) FROM table_name;
AVGSELECT AVG(column_name) FROM table_name;
MAXSELECT MAX(column_name) FROM table_name;
MINSELECT MIN(column_name) FROM table_name;

9. Set Operations

Combine results of two or more queries.

OperationSyntax Example
UNIONSELECT column_name FROM table1 UNION SELECT column_name FROM table2;
UNION ALLSELECT column_name FROM table1 UNION ALL SELECT column_name FROM table2;
INTERSECTSELECT column_name FROM table1 INTERSECT SELECT column_name FROM table2;
EXCEPTSELECT column_name FROM table1 EXCEPT SELECT column_name FROM table2;

10.Views

Command Syntax Example
CREATE VIEW CREATE VIEW view_name AS SELECT column1, column2 FROM table_name;
DROP VIEW DROP VIEW view_name;

11.Indexing

Command Syntax Example
CREATE INDEX CREATE INDEX index_name ON table_name (column_name);
DROP INDEX DROP INDEX index_name;
